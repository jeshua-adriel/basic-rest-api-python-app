# Basic REST API Python App

## Project Overview
This project is a simple Python application that retrieves data from a public REST API and displays it to the user. Our team selected the Joke of the Day App.

## Features
- Fetches a random joke from a public API
- Displays the joke in the terminal
- Includes basic error handling
- Includes unit testing

## API Used
Official Joke API:
https://official-joke-api.appspot.com/random_joke

## How to Run
1. Clone the repository:
   git clone <your-github-repo-link>

2. Open the project folder:
   cd rest-api-python-app

3. Install dependencies:
   pip install -r requirements.txt

4. Run the app:
   python app.py

## How to Run Tests
python -m unittest test_app.py
